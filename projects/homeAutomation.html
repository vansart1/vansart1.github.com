Home Automation System

        I designed a home automation system to control the lights and temperature using a netduino micrcontroller. One of the strengths of this system is its ability to be controlled via various methods. The box housing the electronics has four buttons which can be used to control the lights or adjust the set temperature. The lights and temperature can also be adjusted by visiting the the website or using RF controllers located around the room.

        The home automation system also outputs the results of which lights are on and the current room temperature on its website. The box housing the electronics also contains LCD display that either shows the current date and time, or the current room temperature. The date and time are very accurate as it is regularly fetched and updated from a timeserver.
